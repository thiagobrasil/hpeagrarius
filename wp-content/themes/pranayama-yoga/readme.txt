== Description ==

Pranayama Yoga is modern and clean WordPress theme. It is an ideal choice for Yoga. It is designed with conversion in mind and thus it will increase your lead via phone calls and email enquiries. Pranayama Yoga is mobile friendly and displays content nicely in mobile devices. The theme is translation ready.

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

== Documentation ==
 
 * Home Page Setup
    * To set up the homepage you must first create a new page.
        => Create a Home Page
        => Go to Pages > Add New.
        => Enter the Page Title for the Page. You do not require to enter anything in the content section for creating a home page.
        => Select the Homepage Template from the Page Attributes section.
        => Click Publish.
        
    * Configuring the front Page display setting
        => Go to Appearance>Customize > Default Settings> Static Front Page.
        => Select A static page under Front Page displays
        => Choose the page you just created with homepage template as Front page.
        => Click Save & Publish.

    * Banner Settings
        => Go to Appearance> Customize>Home Page Settings>Banner Settings
        => Check Enable Banner section for it to appear on Homepage.
        => Select a Banner Post.(Featured images of the posts of the selected category will be displayed as banner images. Post Title will appear as Banner Title and Excerpt of the Post will appear on banner as banner description )
            
            * Featured images of posts should be at least 1920 px in width and 435 px in the height for displaying Banner images properly.
            
            * Use regenerate thumbnail plugin to crop all the images if you have already uploaded the Images.
        
        => Click Save & Publish.
        
    * About Section
        => Go to Appearance> Customize>Homepage Settings> About Section  
        => Check Enable About Section to display it in the Homepage.
        => Choose the post for About Section.
            * The post title of the selected post will appear as the section title.
            * The featured image and excerpt of the selected post will be displayed under About  Section.
        => Enter Read More Text 
        => Click Save and Publish.

    * Information Section:
        => Go to Appearance> Customize>Homepage Settings>Information Section
        => Check Enable Information Section to display it on the homepage
        => Select Post One, Post Two, Post Three and Post Four.
            * The Post Title and the content of the selected Post will be displayed under the Information Section .
        => Click Save & Publish.

    * Yoga Classes Section
        => Go to Appearance> Customize> Home Page Settings> Yoga Classes Section .
        => Check  Enable Yoga Classes Section to display it on the Homepage.
        => Enter the Section title
        => Select Post One, Post Two, Post Three, Post Four, Post Five and Post Six.
           * The Featured Image of the selected posts will be displayed under the  section
           * The title of the selected Posts will appear as the Class name.
           * Excerpt of the posts can be used to display Class details such as time and date.
        => Click Save & Publish

    * Promotional Section
        => Go to Appearance> Customize>Homepage Settings>Promotional Section
        => Check Enable Promotional Section to display it on the homepage
        => Enter Promotional  Section  Title and Promotional  Section Description 
        => Enter Call to Action Button Text and  Button Link. 
        => Click Save & Publish.

    * Team Section:
        => Go to Appearance> Customize>Homepage Settings>Team Section
        => Check Enable Team Section to display it on the homepage
        => Enter Team Section Title
        => Select Post One, Post Two  and Post Three  for Team Section.
           The post title  will be displayed as team member name.
           The Featured Image of the selected  posts  will be displayed as Team member’s image
           The content of the selected posts  will displayed  as the team member’s description.
           Excerpt of the post will be displayed as the team member’s  designation.
        => Click Save & Publish. 
    
    * Testimonial Section:
        => Go to Appearance> Customize>Homepage Settings>Testimonial Section
        => Check Enable Testimonial Section to display it on the homepage
        => Select a category for Testimonial  Section.
           The post title under the selected category will be displayed under the testimonial section as client’s name.
           The Featured Image of the posts under the selected category will be displayed as client image
           The content of the posts under  selected category will displayed  under the section.
           Excerpt of the post will be displayed as the client’s designation.
        => Click Save and Publish.

    * Blog Section
        => Go to Appearance> Customize>Homepage Settings>Blog  Section
        => Check Enable Blog  Section to display it on the homepage
        => Enter Blog  Section  Title and Blog Section Description 
        => Click Save & Publish.

    * Reason Section:
        => Go to Appearance> Customize>Homepage Settings>Reason Section
        => Check Enable Reason  Section to display it on the homepage.
        => Select Page for Reasons Section.
           The title and the content of the selected page will be displayed as the section title and content
           The featured Image of the selected Page will be displayed under the Reasons Section.
        => Choose Posts for the Reasons Section.
           The title and the content of the selected post will be displayed as the feature title and content
           The featured Image of the selected Post will be displayed as the Point’s Image.
        => Click Save & Publish.
    
    * Promotional Section Two
        => Go to Appearance> Customize>Homepage Settings>Promotional Section Two
        => Check Enable Promotional Section Two  to display it on the homepage
        => Enter Promotional  Section Two  Title and Promotional  Section Two  Description 
        => Enter Call to Action Button Text and  Button Link. 
        => Click Save & Publish.
